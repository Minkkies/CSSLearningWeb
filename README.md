# CSSLearningWeb

โครงการ **CSSLearningWeb** นี้มีจุดประสงค์เพื่อรวบรวมและสอนความรู้พื้นฐานและเทคนิคการใช้ CSS สำหรับการพัฒนาเว็บไซต์ เหมาะสำหรับผู้เริ่มต้นและผู้ที่ต้องการปรับปรุงทักษะการตกแต่งเว็บไซต์ด้วย CSS โดยเฉพาะ

## รายละเอียดโครงการ

- สาธิตการใช้งาน CSS เพื่อปรับแต่งหน้าเว็บ
- รวมตัวอย่างโค้ดและเทคนิค CSS ตั้งแต่ระดับพื้นฐานถึงระดับสูง
- มีตัวอย่าง UI ที่ใช้ CSS จริง

## ฟีเจอร์หลัก

- สอน CSS ผ่านตัวอย่าง HTML/CSS
- อธิบายแต่ละ property ของ CSS และตัวอย่างการใช้งาน
- แนะนำแนวทางการจัดโครงสร้าง CSS ในโปรเจกต์ขนาดเล็กถึงขนาดกลาง
- ใช้ตัวอย่างโค้ดเพื่อการเรียนรู้แบบลงมือปฏิบัติจริง

## วิธีเริ่มต้นใช้งาน

1. Clone โปรเจกต์ไปยังเครื่องของคุณ:
   ```bash
   git clone https://github.com/Minkkies/CSSLearningWeb.git
   ```
2. เปิดไฟล์ `.html` ด้วย Web Browser ที่คุณต้องการ
3. แก้ไข หรือทดลอง CSS ในโฟลเดอร์ที่เกี่ยวข้อง
