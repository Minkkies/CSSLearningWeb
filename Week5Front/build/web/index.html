- Internal CSS (การใช้CSS ภายในแท็ก style ใน head)

Result Internal 

<head>
  <meta charset="utf-8">
  <title>CSS Internal</title>
  <style>
    body { font-size: 18px }
    p { font-style: italic }
    span {
      background-color: #ddd; 
      font-weight: bold;
    }
  </style>
</head>

<body>
  <span>การแยกหน้าที่ระหว่าง HTML และ CSS คือ</span>
  <p>HTML ใช้สำหรับการจัดโครงร่างของเว็บเพจ</p>
  <span>CSS ใช้สำหรับการจัดรูปแบบของเว็บเพจ</span>
</body>

- External CSS (การใช้ไฟล์ CSS ภายนอก ใน head )

- <embed> คือ แท็ก HTML ที่ใช้สำหรับ ฝัง (embed) เนื้อหาหรือไฟล์มัลติมีเดียภายนอก เช่น: ไฟล์วิดีโอ , ไฟล์เสียง , PDF

Result Embedded 

<head>
    <meta charset="UTF-8">
    <meta name=".." content="...">
    <title>Style Sheets</title>
    <style type="text/css">//External

        em { //Embeded
            background-color:#8000FF;
            color:white;
        }

        h1 { 
            font-family:Arial, sans-serif;
        }
        p {
            font-size:18pt;
        }
        .blue {
            color:blue;
        }
    </style>
</head>

<body>
    <h1 class="blue">A Heading</h1>
    <p>Here is some text. Here is some text. Here
        is some text. Here is some text. Here is 
        some text.</p>      
    <h1>Another Heading</h1>        
    <p class="blue">Here is some more text.
        Here is some more text.</p>
    <p class="blue">Here is some <em>more</em>
        text. Here is some more text.</p>  
</body>

- Div & P & Element :
<div> จะขึ้นบรรทัดใหม่ นิยมใช้เป็นเหมือนกล่องไว้ครอบเพื่อแต่ง <style>
<span> จะไม่ขึ้นบรรทัดใหม่ ส่วนตัวเอาไว้ใช้คู่กับ <p> การณีอยากแต่งแค่บางส่วนของประโยค

Result Div , P, Span 

<head>
    <meta charset="utf-8">

    <style>
        div + p { font-size: 30px; }
    </style>
</head>

<body>
    <div>div</div>

    <p>second p</p>
</body>

Selector & Style Precedence

- Child:
#div1 > p คือ Child Selector หมายถึงจะเลือก p ที่เป็น ลูกโดยตรง ของ #div1 เท่านั้น
ดังนั้น:
✅ first p จะถูกปรับขนาดตัวอักษรเป็น 30px
❌ second p ไม่โดนเพราะอยู่ใน #div2 ซึ่งเป็นลูกของ #div1 แต่ p นั้นไม่ใช่ลูกโดยตรง

Result Child 

<head>
    <meta charset="utf-8">
    <title>CSS Child Selector</title>
    <style>
        #div1 > p { font-size: 30px; }
    </style>
</head>

<body>
    <div id="div1">
       <p>first p inside #div1</p>
        <div id="div2">
            <p>second p inside #div2</p>
        </div>
    </div>
</body>

- Class ใช้กำหนดคลาสของแต่ละส่วนเช่นในตัวอย่าง
.bold → ตัวหนา
.italic → ตัวเอียง
.underline → ขีดเส้นใต้
.mark → พื้นหลังสีเทา

Result Class 

<head>
  <meta charset="utf-8">
  <title>CSS Class Selector</title>
  <style>
    .bold{ font-weight: bold; }
    .italic { font-style: italic; }
    .underline { text-decoration: underline; }
    .mark { background-color: lightgray; }
  </style>
</head>

<body>
  <p class="bold">คติสอนใจ</p>
    <div class="italic">อย่านอนตื่นสาย</div><br>
    <div class="underline">อย่าอายทำกิน</div><br>
    <div class="mark">อย่าคอยวาสนา</div>
</body>
