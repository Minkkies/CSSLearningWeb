What I Have Learn About CSS

- CSS stands for Cascading Style Sheets

- CSS describes how HTML elements are to be displayed on screen, paper, or in other media

- For meaning of Element you can read more in HTML and CSS link in sidebar

Introduction to CSS

- Inline CSS (การใช้ CSS ภายในแท็ก HTML โดยตรงใน body )

Result Inline 

<body>
    <p>Here is some text</p>
    <p style="font-size: 20pt">Here is some more text</p>
    <p style="font-size: 20pt;color: #0000FF">Even more text</p>
</body>

- External CSS (การใช้ไฟล์ CSS ภายนอก ใน head )

Result External 

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Style Sheets</title>
    <style type="text/css">
        em {
            background-color:#8000FF;
            color:white;
        }
        h1 {
            font-family:Arial, sans-serif;
        }
        p {
            font-size:18pt;
        }
        .blue {
            color:blue;
        }
    </style>
</head>

<body>
    <h1 class="blue">A Heading</h1>
    <p>Here is some text. Here is some text. Here
        is some text. Here is some text. Here is some
        text.</p>
    <h1>Another Heading</h1>
    <p class="blue">Here is some more text.
        Here is some more text.</p>
    <p class="blue">Here is some <em>more</em>
        text. Here is some more text.</p>
</body>
